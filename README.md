This website has been deployed at vercel

[Click here to view the website](https://nscc-task.vercel.app/)

To deploy locally -


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
